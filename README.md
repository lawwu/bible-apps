# Every Word Entwined

An interactive explorer of how the Bible cross-references itself — walk verse
to verse, word to verse, or topic to verse through ~340,000 reader-voted
connections.

**Live: https://lawwu.github.io/bible-apps/**

Made with [Open Bible](https://www.openbible.info/) data and Claude Fable 5 (High).
