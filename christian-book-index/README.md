# The Christian Book Prize Index

This folder is the built static output (Next.js `output: "export"`) of The
Christian Book Prize Index, a ranked, searchable catalog of award-winning
Christian books aggregated across six prize programs and marked where
traditions cross-recognize a title. The source and data pipeline live at
`~/github/vibe-apps/christian-book-index`; nothing here is hand-edited. To
rebuild, run `NEXT_BASE_PATH=/bible-apps/christian-book-index npm run build`
in that project and re-copy its `out/` directory over this one.
